Python Calculator

This is a simple calculator made in Python while I was learning the basics.

It can do:

- addition
- subtraction
- multiplication
- division
- exponent
- floor division
- modulus

It also handles divide by zero and lets you use the previous result for another calculation.

---

What I learned from this

- taking input from user
- if else conditions
- working with numbers
- handling simple errors
- building logic step by step
